Overview

Machine learning is an application of artificial intelligence (AI) that provides systems the ability to automatically learn and improve from experience without being explicitly programmed. This project implements some basic methods/algorithms for machine learning. It consists of Linear Regression, Binary Classification, Feature cross, Gradient Descent with Momentum.

Goal

Providing a simple Machine Learning library that anyone can include to their program.

Description

        This main file of this project is named  ML.h  which has all the algorithms mentioned above. There is also another file named csv_reader.h which is useful when the training data is in a CSV file format. The ML.h file includes the csv_reader.h file so the user only has to include the ML.h file only. Using the functions of the csv_reader.h file we can open a  CSV file consisting of Numeric values only specifying the label column and it will convert the training features to a 2D array of floating-point values and the labels to a 1D array of floating-point values. After that, the user can perform training and predicting by just calling the suitable functions and passing valid arguments. After Training the user can save the model in a file.

        void read_csv (CSV *csv,  char *filePath )

This function takes a pointer to the CSV structure and the path of the file to be opened. It opens the file in read mode and assigns the file pointer to the file pointer variable of the CSV object.

        char** toArray ( char *head )

                            This function takes a character pointer which has characters separated by comma ( , ) and splits the array at the comma and converts it to a 2D array of characters and return it.

        

        void makeTableFLoat(CSV *csv,  int batch_size, int  label_column )

                This function first uses the CSV  ‘s file pointer variable and reads the file line by line. First, it skips the header, then reads a line, store it to a 1D array of characters.

Then it uses the toArray function to convert it to 2D array of characters in which each row is string of the Numeric values of the file. Then it uses the strtof function to convert each row of the 2D array to a floating-point value. Then it stores each floating-point value in the row of the floatTable  2D array of CSV  structure. The label of each row is stored in floatLabels 1D array of CSV structure. It reads a batch_size number of lines from the file.

void initialize_weights(Model *M,  int num_W )

        This function takes the Model structure and initialize  num_W  number of weights for the single-layered neuron. It uses time function to get the current time in milliseconds and performs some arithmetic operations to generate pseudo-random numbers. These numbers then inserted into the W array of Model structure.

            void compile(Model *M,  int num_X, int  size_X_per, char *loss_fn_choice, int show_cost )

        The num_X  is basically the number of training examples.

        The size_X_per is the number of features in each training example.

        The loss_fn_choice is a string of the name of the loss function to be used.

        The show_cost determines whether to show the cost while training or not.

         This function takes these values and assigns them to the corresponding variables of the Model structure.

        void forward_propagate(Model *M, float  **X, int epoch_no )

        This function performs forward propagation of the neuron. It multiplies the features of each example  X[i][j]  with the corresponding weights M->W[i]. Then adds the bias term M->b  and passes it to the activation function (sigmoid). Then it calculates the cost by using the loss function (binary cross-entropy) and displays the output of epoch number, cost, accuracy. It also cleans the display before showing the output.

        void back_propagate(Model *M, float  **X, float *Y)

                This function performs backpropagation and updates the weights and bias.

It iterates through each training example and calculates gradients of weights and bias. It also adds momentum to gradients by using a proper algorithm. Then it updates the weights by subtracting the gradients multiplied by the learning rate. All changes saved in the Model structure itself.

        void classification_train(Model *M, float  **X,  float  *Y,  float  learning_rate,  int epochs )

        This function is used when the user wants to perform a simple classification task.

It sets the learning rate and calls forward_propagation and back_propagation function epochs number of times inside a loop.

        void forward_propagate_linearRegressor(Model *M, float  **X, int epoch_no )

        This function is used when the user wants to perform a simple linear regression task. It works similarly to forward_propagation function but it does not use an activation function to add non-linearity. The loss function is also different (mean squared error)  here.

        void back_propagate_linearRegressor(Model *M, float  **X, float  *Y)

        This function works similarly to back_propagation function but the algorithm to compute gradients is different since in linear regression there is no activation function and the loss function is mean squared error.

        void linearRegressor_train(Model *M, float  **X,  float  *Y,  float  learning_rate,  int epochs )

        This function sets the learning rate and calls the  forward_propagate_linearRegressor

And back_propagate_linearRegressor function epochs number of times inside a loop.

        void feature_cross_and_train_classification(Model *M, float  **X,  float  *Y,  float  learning_rate,  int epochs , char *type)

        This function performs feature cross and trains at the same time. A feature cross is a synthetic feature formed by multiplying (crossing) two or more features. Crossing combinations of features can provide predictive abilities beyond what those features can provide individually.

Currently, this function can perform two types of feature cross: Squaring and Taking Sine.

The type string is the type of feature cross user wants.

In the Squaring: The function takes each example and makes another copy by squaring them and then join it to the example. It also reinitializes the weights to match proper dimensions.

Taking Sine: The function takes each example and makes another copy by taking the sine of them and then join it to the example. It also reinitializes the weights to match proper dimensions.

Then it calls the forward_propagate and back_propagate epochs number of times inside a loop.

        float predict(Model *M,  float *X)

This function takes the Model and a single test array of features and returns the resulting prediction after training.

Classification

#include "ML.h"

int main(){

    int batch_size = 78;  // reads the  csv file in batches

    int label_column = 2; //label column is at postion 2 (counting from 0)

    char *filePath = "./circle.csv";  //path to the csv file

    int features_len = 2;  // since there are 2 feature inputs in the csv

    int num_features = 78;  // there are 78 features to train

    float learning_rate = 0.01 ;  //for backpropagation

    int num_epochs = 5000;

    float test[][2] = {{100,3},{3,1.8}, {0,2}, {3,4.5}, {1.9,3.5}, {6,6}, {2.7, 2.5}}; //testing samples

    CSV csv; 

    read_csv(&csv, filePath);  //reading the csv file

    makeTableFloat(&csv, batch_size, label_column);      // making the table of features and labels  

    Model model;   

    initialize_weights(&model, features_len); 

    model.beta = 0.9; // momentum for gradent discent (optional)

    compile(&model, num_features, features_len, "binary_crossentropy", 1 );

    feature_cross_and_train_classification(&model, csv.floatTable, csv.floatLabels, learning_rate, num_epochs, "X^2" );

    for(int i = 0; i < 6; i++){

        float pred = predict(&model, test[i]);

        printf("\n(%f,%f)-> prediction is: %f\n", test[i][0], test[i][1], pred);

    }

  return 0;

}
